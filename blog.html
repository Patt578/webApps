Java Collections API

Today I finished researching concepts surrounding Java's built in Collections Library, this includes the Map interface, sets, and of course all kinds of Lists which I've been using for some time now.

My current goal now is to master my knowledge of the library by making a simple text adventure game that uses Maps as a framework for locations and exits within the game. As well as use sets to group together similar type monsters. Since Maps are built with keys assigned to values which is perfect for locations and exits. And then regarding sets, this will allow me to group Monster objects in no order for random spawns upon entering a room.
